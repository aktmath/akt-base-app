# AKT Base App

Base Sepolia testnet üzerinde çalışan basit bir counter uygulaması.

## Özellikler
- Next.js + OnchainKit
- Foundry smart contracts
- Base Sepolia testnet entegrasyonu

## Kurulum
```bash
npm install
npm run dev
```
